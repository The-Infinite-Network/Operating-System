# Privacy Policy — Lil L

**Last updated:** March 16, 2026
**Operator:** Big_L_OKC

---

## 1. Overview

Lil L is a private Discord bot operated by Big_L_OKC. It is not publicly distributed and is only active in servers where it has been explicitly authorized by the operator.

---

## 2. Data We Collect

Lil L does **not** collect, store, log, or retain any personal data.

When performing its functions, the bot transiently accesses the following Discord data solely to fulfill a request in real time:

- **Message content** — to read or send messages in authorized channels
- **User identifiers** (username, user ID) — as provided by Discord's API in message objects
- **Server and channel identifiers** — to route requests to the correct channel

None of this data is written to a database, log file, or any external service. Once a request completes, no record is kept.

---

## 3. Data Sharing

Lil L does not sell, share, or transmit any user data to third parties. The bot communicates only with:

- **Discord's API** — to read and send messages per authorized requests
- **The operator's local MCP server** — an internal service running on the operator's own infrastructure

---

## 4. Data Retention

No data retention. The bot processes requests in memory and retains nothing after each request completes.

---

## 5. Who Has Access

This is a private bot. It is not listed on any bot directory and cannot be added by third parties. Only the operator controls which servers and channels the bot can access.

---

## 6. Children's Privacy

Lil L is not directed at children under 13. Because the bot collects no data, it does not knowingly collect information from any user, including minors.

If this policy changes, the updated version will be posted here with a revised date. Continued use of the bot constitutes acceptance of the updated policy.
